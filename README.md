# FVTT-AspectDices
Adds a new type of dice: the Aspect Die.  
Comes with 'Dice So Nice!' support.  
Usage: 
```
/r 1db 1de 
game.aspectmod.macros.RollAspects()
game.aspectmod.macros.RollAspects2()
```
